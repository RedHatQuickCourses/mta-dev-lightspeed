= Modernizing Java Applications with Red Hat Developer Lightspeed for MTA
:navtitle: Home

== Introduction

Welcome to this introductory training on **_Red Hat Developer Lightspeed for Migration Toolkit for Applications (MTA)_**.

This course is designed to introduce **Developer Lightspeed**, explain its role in application modernization, and demonstrate how it works in combination with **Red Hat MTA** to accelerate Java application migration.

== What's in this course?

This training combines a brief conceptual overview with a hands-on, lab-driven use case based on the **Coolstore** application. You will begin by understanding the purpose and capabilities of **Red Hat Developer Lightspeed for MTA**, followed by practical hands-on exercises that demonstrate its value in a real-world modernization scenario.

Duration: 1 hours

== Objectives

On completing this course, you should be able to:

* Understand the purpose and key capabilities of **Red Hat Developer Lightspeed for MTA** and how it assists developers during application modernization.

* Deploy a legacy **Coolstore** application running on **JBoss EAP 7**, built using the javax-based Java EE APIs.

* Configure the **MTA extension in Visual Studio Code** and enable **Developer Lightspeed** using a **Generative AI model (Llama) provided through Red Hat Model-as-a-Service (MaaS)**.

* Analyze the legacy application source code using MTA and Developer Lightspeed to identify migration issues and receive guided refactoring recommendations.

* Apply the suggested changes to migrate the application from the **javax** namespace to the **Jakarta EE** platform.

* Deploy and validate the migrated application on **JBoss EAP 8**.

== Prerequisites

This course assumes that you have the following prior experience:

* Working knowledge of Java SE
* Basic understanding of Java EE / Jakarta EE concepts
* Familiarity with legacy javax APIs is helpful
* Red Hat Openshift Platform
