= Lab 4: Deploying the Migrated App on JBoss EAP 8

In the final section, you apply the recommended changes to migrate the **Coolstore** application to **Jakarta EE**.
You rebuild the container image using **JBoss EAP 8**, deploy it on OpenShift, and validate that the modernized application functions as expected.

This step completes the migration journey, demonstrating how a legacy enterprise Java application can be successfully transformed and redeployed on a modern, supported runtimeâ€”without disrupting the underlying container platform.

== Instructions

* Navigate to the cloned legacy application code:
+
[source,subs="verbatim,quotes",bash]
----
**git clone https://github.com/opentraining-labs-customizations/getting-started-migrated.git**

**cd getting-started-migrated**
----

* Substitute <GUID> with your Openshift cluster id in below file:
+
[source,subs="verbatim,quotes",bash]
----
**vi coolstore/src/main/webapp/keycloak.json**
{
    "realm": "eap",
    "auth-server-url": "https://keycloak-coolstore-mono.apps.cluster-**<GUID>**.dynamic.redhatworkshops.io/",
    "ssl-required": "external",
    "resource": "eap-app",
    "public-client": true,
    "confidential-port": 0
}
----

* Build the code to reflect the changes in jar file using maven:
+
[source,subs="verbatim,quotes",bash]
----
**mvn clean package -pl coolstore -am**
----
+
Note the file is build at below location:
+
[source,subs="verbatim,quotes",bash]
----
**ls coolstore/target/ROOT.war**
----

* Copy the application war file as shown below to the new cloned application deployment repository:
+
[source,subs="verbatim,quotes",bash]
----
**ls**
getting-started  mta-dev-lightspeed

**cp getting-started-migrated/coolstore/target/ROOT.war  \** **mta-dev-lightspeed/eap8_coolstore/00-eap8-coolstore-image**
----

* Next, let's deploy the application.

[source,subs="verbatim,quotes",bash]
----
**cd mta-dev-lightspeed/eap8_coolstore/00-eap8-coolstore-image/**

buildconfig.build.openshift.io/coolstore-eap81 created
imagestream.image.openshift.io/coolstore-eap81 created
[redhatsystem 00-eap8-coolstore-image]$
