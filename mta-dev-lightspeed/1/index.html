Modernizing Java Applications with Red Hat MTA & Developer Lightspeed

Introduction

Welcome to this quick course on the Modernizing Java Applications with Red Hat MTA & Developer Lightspeed.

In this hands-on lab training, we shall see all these features of Red Hat Migration Toolkit for Applications (MTA) along with Developer Lightspeed in action.

What’s in this course?

A hands-on lab-oriented training demonstrating how to migrate a real-world enterprise Java application using Red Hat Migration Toolkit for Applications (MTA) and Developer Lightspeed, transforming a legacy javax-based application deployed on JBoss EAP 7 into a modern jakarta-based application running on JBoss EAP 8.

Duration: 1 hours
