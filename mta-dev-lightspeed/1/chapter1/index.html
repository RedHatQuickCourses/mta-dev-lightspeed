Red Hat Developer Lightspeed for MTA

Introduction

  • Red Hat Developer Lightspeed for migration toolkit for applications speeds up application refactoring with AI-generated code that is tailored to your enterprise’s applications.

  • It is an add-on to Red Hat MTA that intelligently automates source code refactoring directly within a developer’s IDE.

  • This allows organizations to modernize applications more efficiently and accelerate adoption of Red Hat OpenShift and cloud-native technologies, improving the overall return on investment for modernization initiatives.

  • Developer Lightspeed provides clear explanations and recommended code changes during the refactoring process.

  • Developers can review these suggestions and apply fixes with a single click. The tool supports both out-of-the-box migration paths as well as custom rules, enabling teams to modernize applications that use organization-specific or proprietary technologies.

  • Red Hat MTA primarily supports application assessment and re-platforming, while Developer Lightspeed for MTA focuses on code refactoring use cases. Together, these tools provide comprehensive support for analyzing applications, planning migrations, and updating source code.

Challenges of modernizing applications

  • Security concerns: As migrators refactor applications to use cloud-native technologies, developers (migrator) may accidentally create new security vulnerabilities.

  • Complexity of legacy systems: Legacy systems often lack compatibility with modern platforms, may have tightly coupled and hidden dependencies, or may be an unfamiliar codebase to the migrator - posing the risk of unintentionally altering or breaking application functionality.

  • Competing priorities: Enterprises need to balance meeting market challenges and opportunities with the time and costs it takes to improve existing systems, which can be significant. As a result, stakeholders may prioritize new features or applications over improving existing code of their whole application portfolio.
